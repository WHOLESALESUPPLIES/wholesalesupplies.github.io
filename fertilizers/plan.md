   Пиролиз — это процесс термического разложения органических материалов при отсутствии кислорода. Этот метод также может быть использован для сжигания бумажных отходов, включая банкноты, с получением минимального количества вредных выбросов.

   **Пример оборудования**:
   - **Пиролизные установки для отходов**. Эти установки могут эффективно перерабатывать бумажные отходы, разрушая их при высоких температурах (от 400°C до 700°C).
   
     **Основные характеристики**:
     - Процесс сжигания в безкислородной среде.
     - Получение твердого остатка (золы) и газов, которые могут быть использованы для генерации энергии.
     - Система фильтрации и очистки газов от токсичных веществ.

   **Пример производителей**:
   - **Pyrocal** — компания, предлагающая пиролизные установки для переработки различных видов отходов, включая бумагу.
   - **Carbolite Gero** — производит оборудование для пиролиза и высокотемпературного обжига, включая системы для термической переработки отходов.

   **Пример оборудования**:
   - **Фильтры с активированным углем** — используются для удаления вредных химических веществ и газов из выбросов.
   - **Электрофильтры** — для фильтрации твердых частиц и токсичных загрязнителей в газах.

### Заключение
Для утилизации денежных купюр методом сжигания требуется специализированное оборудование, которое должно обеспечивать не только безопасное сжигание, но и фильтрацию вредных выбросов. Важными элементами процесса являются:
- Печи для сжигания отходов.
- Пиролизные установки для безопасного перераспределения энергии.
- Установки для фильтрации газов и очистки выбросов.
- Шредеры для измельчения банкнот.

В результате использования таких технологий, можно обеспечить безопасную, эффективную и экологически чистую утилизацию денежных купюр.
